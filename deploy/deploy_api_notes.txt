0.  Decide upon target deployment:

    0.1 ENV = prod | test

    0.2 APP_ID = 1 | 2

    0.3 APP_NAME = ENV + "_api" + APP_ID, e.g. prod_api3

    0.4 APP_VERSION = 0.8.6.2

1.  Login into webfaction web control panel

    1.1 Create new mod_wsgi 3.3 / python 2.6 application : APP_NAME

        1.1.1   E.G. - prod_api3

        1.1.2   IMPORTANT - Note the application port (PORT_NO)   (e.g. 20415)

    1.2 Create new postgres db : "esdoc_" + APP_NAME

        1.1.1   E.G. - esdoc_prod_api3

        1.2.2   IMPORTANT - Note the db password (DB_PWD)  (e.g. pU8WAd4U)

2.  SSH into webfaction server

    2.1 Run install script : scripts/api_install.sh ENV APP_ID APP_VERSION PORT_NO DB_PWD

        2.1.1   E.G. - scripts/api_install.sh prod 3 0.8.6.2 20415 pU8WAd4U

        2.1.2   IMPORTANT - Answer 'y' if asked to replace db file:
                    replace /home/esdoc/webapps/APP_NAME/tmp/deploy/db? [y]es, [n]o, [A]ll, [N]one, [r]ename:

        2.1.3   IMPORTANT - enter db password when prompted

    2.2 Restart apache : webapps/APP_NAME/apache2/bin/restart

        2.2.1   E.G. - webapps/prod_api3/apache2/bin/restart

3.  Login back into webfaction web control panel

    3.1 Redirect target website to new app

        3.1.1   Select website, e.g. prod_api

        3.1.2   Delete existing app reference.

        3.1.3   Add new app reference

        3.1.4   Save changes

        3.1.5   NOTE - normally takes about 1 minute to take effect
